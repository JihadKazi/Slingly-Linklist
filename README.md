Linked List Operations in C

A simple C program demonstrating singly linked list operations, including list division, unique merging, and traversal.

Features :
Create two linked lists
Divide the first list into:
X: Elements greater than 25 or divisible by 4
Y: Remaining elements
Merge List X with List 2
Remove duplicate values during merging
Display all resulting linked lists

Technologies  :
C Programming
Singly Linked List
Dynamic Memory Allocation
Functions

Compile & Run :
gcc main.c -o linked_list
./linked_list

Concepts Covered :
Linked Lists
Dynamic Memory Allocation (malloc)
Traversal
Conditional Data Separation
Duplicate Removal
List Merging
